API Documentation

The Aureus Institutional API provides low-latency access to global market data, advanced AI execution protocols, and real-time portfolio analytics. Designed for high-frequency trading and quantitative research.

GET

/v1/market/prices

Retrieves real-time bid/ask prices for a specified asset class or individual ticker symbol. Recommended polling frequency for institutional tier is 50ms.

Query Parameters

Name
Type
Status
Description
symbol
string
Required
The ticker symbol (e.g., BTC-USD).
depth
integer
Optional
Order book depth. Default is 10. Max is 1000.
POST

/v1/orders/execute

Example Request 
curl -X GET \
  "https://api.aureus.terminal/v1/market/prices?symbol=BTC-USD&depth=5" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "X-Aureus-Client: v2.1.0"
Example Response (200 OK) 
{
  "status": "success",
