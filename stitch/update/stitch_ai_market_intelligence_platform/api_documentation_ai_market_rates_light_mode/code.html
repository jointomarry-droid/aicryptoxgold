Market Data

Retrieve real-time and historical market data across various asset classes. This endpoint provides high-fidelity pricing information necessary for institutional algorithmic trading strategies.

Get Asset Prices GET

https://api.aureus.com/v1/market/prices

Fetches the current top-of-book pricing for a specified array of ticker symbols. Requires read-level API key permissions.

Query Parameters

Name Type Status Description
symbols array[string] Required Comma-separated list of ticker symbols (e.g., AAPL,MSFT). Max 50 per request.
exchange string Optional Filter by specific exchange routing (e.g., NASDAQ, NYSE). Defaults to consolidated tape.
format string Optional Response format. Accepts json or csv. Defaults to json.

Get Historical Order Book GET

https://api.aureus.com/v1/market/book/historical

Retrieve L2 depth of book snapshots at specific historical timestamps. Useful for backtesting execution algorithms.

Example Request 
curl --request GET \
  --url 'https://api.aureus.com/v1/market/prices?symbols=AAPL,MSFT' \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Accept: application/json'
